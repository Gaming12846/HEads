# Heads version 1.1
A simple plugin to get heads from players and things in Minecraft

Features:
- Get the head from players and things with a command
- Craftrecipes to craft heads from players and things
- Customizable Config

Commands:
- /gethead or /gethead player

Aliases:
- /gethead : [getheads, gh]

Permissions:
- heads.gethead
- heads.headrecipe
