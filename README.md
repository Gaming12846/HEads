# Heads v1.1.1
A simple plugin to get heads from players and things in Minecraft

Native Minecraft Version: 1.17

Features:
- Config
- Multilingual support (Config)
- Get heads
- Craft heads

Commands:
- /gethead or /gethead player

Aliases:
- /gethead : [gh]

Permissions:
- heads.gethead
- heads.headrecipe
